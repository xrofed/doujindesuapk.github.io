Cara Menginstal

  1. Download Apk-nya melalui tombol di atas.
  2. Pastikan memori penyimpanan kamu tidak penuh.
  3. Aktifkan "Sumber tidak dikenal" melalui pengaturan keamanan ponsel.
  4. Instal Apk melalui file manager bawaan atau aplikasi ZArchiver/X-plore.
  5. Apabila ingin update, instal langsung (timpa). Jika gagal, hapus (uninstall) versi lama terlebih dahulu.
  6. Apabila muncul peringatan Play Protect, klik "Detail" lalu pilih "Tetap Install".
